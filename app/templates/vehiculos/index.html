Bienvenido al Taller de Latonería

Este sistema permite gestionar vehículos, clientes y procesos de reparación de forma eficiente.

¿Qué puede hacer este sistema?

  • Vehículos: Registrar autos, actualizar información y gestionar reparaciones.
  • Clientes: Registrar clientes, actualizar datos y ver historial.
  • Historial: Visualizar y consultar operaciones realizadas.

Objetivo del proyecto

Este sistema fue desarrollado como parte del proyecto académico para la materia Desarrollo de Software.
